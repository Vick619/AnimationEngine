Animation Engine Start Up Tutorial

Top Bar

The top bar is the main control console for the animation engine, allowing you to change the speed, size and opacity of the animated object.
The button next to the input boxs are the animation controls; such as play, stop, restart and reset.

Using images

Using images in the engine si fairly straight forward. Firstly make sure that all the images that you would like to use are saved somewhere on your computer, as using image URL's from the internet will not work.

Once the images have been saved on the computer, just drag and drop them into the boxes in which they will be used. i.e. Objects into the objects box and backgrounds into the background box.

Almost there...

Once all images are in their places, select the ones you would like to use by clicking on them. You cn at any point change your images by clicking on diffrent image.
Once you have the images that you want to use, start by setting a desired size and speed for the animation.
At this point everything is all set up and you are able to start creatung keyframes for the motion path of the object/image.
